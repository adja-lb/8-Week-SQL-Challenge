CREATE SCHEMA Week1_Dannys_Dinner;

## :nerd_face: What I learnt :sunglasses:

* Filtering, ordering, aggregation
* Joins, subqueries, and complex table relationships
* Common Table Expressions (CTEs) and recursive queries
* Window functions, ranking, moving / cumulative metrics
* Handling duplicates, cleaning and exploring data
* Date / timestamp manipulation, type conversions
* Creating entity-relationship diagrams (ERDs) and understanding schema design
* Realistic data-analysis workflow: schema setup, data load, querying, insight generation
